# GST Billing & Inventory ERP (mock)

This workspace contains a Vite + React mock of a multi-tenant GST Billing, Inventory and Accounting application.

Run locally:

```powershell
npm install
npm run dev
# In another terminal to serve mock DB:
npm run json-server
```

Key pages:
- `/sale/new` - dynamic sales invoice form with dense UI
- `/reports` - Day Book and Balance Sheet sample
- `/settings` - app-wide settings panel
